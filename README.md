# rh-crypto-rules

## Introduction

Overlay repo with custom rules that covers internal use cases (those that won't be accepted in upstream).

By now the main use case that aims to be faced here is to detect a library or framework usage.

The image with the semgrep-rules is being publicly pushed <docker://quay.io/exd-guild-security/rh-crypto-scanner-public/rh-crypto-rules:latest>.

The test framework for this repo is at <https://github.com/scanoss/crypto_rules.git>
